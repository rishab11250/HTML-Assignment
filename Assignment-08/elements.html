HTML Elemts showcase – Inline vs Block

Introduction:
In HTML, elements are broadly divided into two categories: block-level and inline. Block elements take up the full width, while inline elements only take as much space as needed.

[Block Elements][Inline Elements][Comparison]

Block Elements

1. Example:

<p>This is a Paragraph</p>

Explanation: The <p> tag defines a block of text that always starts on a new line

2. Example :

<h1>This is a heading<h1>

Explanation: Heading are block-level elements that take the full width

3. Example:

<blockquote>"Coding is the literacy"</blockquote>
Explanation: Blockquote is a block container for quotations.

4. Example:
<ul>
          <li> Apple </li>
          <li> Mango </li>
</ul>
Explanation: List is a block element for making unordered list and ordered list.

5. Example:
<div>This is a division.</div>
Explanation: The <div> tag is a generic container used to group other elements for styling or scripting. It is a quintessential block-level element.

6. Example:
<form>... input elements ...</form>
Explanation: The <form> tag is used to create a container for user input. It is a block-level element.

7. Example:
<section> <h2>Chapter 1</h2> </section>
Explanation: The <section> tag defines a thematic grouping of content. It is a semantic block-level element.

8. Example:
<header> <h1>Main Page Title</h1> </header>
Explanation: The <header> element represents introductory content, typically a group of introductory or navigational aids. It is a block-level element.

Inline Elements

1. Example:
This word is <span style="color:blue;">blue</span>.
Explanation: The <span> tag is a generic inline container used to group or style a part of the text without creating a new line.

2. Example:
<a href="#">Click me</a>
Explanation: The <a> (anchor) tag defines a hyperlink. It flows within the text and only takes up the space of its content.

3. Example:
This text is <strong>important</strong>.
Explanation: The <strong> tag indicates text with strong importance, seriousness, or urgency. It is typically displayed as bold and remains inline with the surrounding text.

4. Example:
This text is <em>emphasized</em>.
Explanation: The <em> tag marks text with stress emphasis. It is usually displayed in italics and does not break the flow of text.

5. Example:
Here is an image: <img src="image.png" width="50">
Explanation: The <img> tag embeds an image in the document. It is an inline element, meaning text can wrap around it.

6. Example:
Enter your name: <input type="text">
Explanation: The <input> tag is used to create interactive controls for web forms. By default, it is an inline element.

7. Example:
First line.<br>Second line.
Explanation: The <br> tag inserts a single line break. It is an empty inline element used to force the subsequent content to a new line without creating a new paragraph block.

8. Example:
Use the <code>document.getElementById()</code> method.
Explanation: The <code> tag is used to define a piece of computer code. It is displayed inline with a monospace font.
